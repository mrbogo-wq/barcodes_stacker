ENGLISH

In the same repository, add a folder named "barcodes", in which add folders "batch1" and up (ex: "batch2", "batch3") if you have more chunks of generated codes.
The barcode gifs use special naming and indexing, ex, "barcode (0)". Incrementing from index 0.

The output is a neat PDF file you can send to print right away! The ordering is left to right, then top to bottom.

RUSSIAN

В том же репозитории добавьте папку с именем "barcodes", в которую, если у вас есть несколько папок сгенерированных кодов, добавьте папки "branch1" и выше ("batch2", "batch3").
GIF-файлы со штрихкодами используют специальную систему именования и индексации, например, "barcode (0)". Индекс увеличивается с 0.

В результате получается PDF-файл, который можно сразу же отправить на печать! Порядок элементов: слева направо, затем сверху вниз.
